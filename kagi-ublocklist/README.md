# Kagi uBlocklist

Sync user Kagi blocking list with the one provided by https://github.com/popcar2/BadWebsiteBlocklist

## Usage

1. Retrieve Kagi Token at `https://kagi.com/settings?p=user_details` -> `Session Link` -> `https://kagi.com/search?token=KAGI_TOKEN`
2. Put it in `KAGI_TOKEN` inside `sync.sh`
3. Run `./sync.sh`
4. Done.
